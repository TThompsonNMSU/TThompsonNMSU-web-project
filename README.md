# TThompsonNMSU-web-project
# GreenTech Solutions Website

## Project Description
This project is a responsive multi-page website created for ICT 280. The website focuses on sustainability and clean technology solutions while demonstrating front-end web development skills using HTML, CSS, and JavaScript.

The site was designed to provide a modern user experience with responsive layouts, accessibility considerations, and interactive features.

---

## Live Site Link
GitHub Pages URL:

https://tthompsonnmsu.github.io/TThompsonNMSU-web-project/final/index.html

---

## Features
- Responsive design for desktop, tablet, and mobile devices
- Multi-page website navigation
- Custom HTML structure and semantic elements
- CSS styling with layout customization
- JavaScript interactivity and dynamic features
- Accessibility considerations including readable contrast and organized navigation
- Mobile-friendly navigation and layout adjustments using media queries
