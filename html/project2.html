OSI Model Layers

The OSI model has seven layers. Each layer performs a specific function in the process of network communication.

OSI Layer Breakdown

Layer # Name Function
7 Application Provides network services to applications
6 Presentation Formats and encrypts data
5 Session Manages connections between devices
4 Transport Ensures reliable data delivery
3 Network Handles routing and addressing
2 Data Link Controls error detection and frame transmission
1 Physical Transmits raw bits over physical medium
